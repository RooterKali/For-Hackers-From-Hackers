## 👋 What is For Hackers, From Hackers?

A community-driven knowledge base for bug bounty hunters and security researchers, curated from resources, techniques, tools, research, and insights shared by hackers across the security community.

The goal is simple: bring useful security knowledge scattered across different platforms into one organized and accessible place.

## 🧠 Why This Project?

Useful security knowledge is scattered across social platforms, write-ups, repositories, discussions, and research shared by hackers around the world. **For Hackers, From Hackers** aims to curate and organize those resources into one accessible knowledge base for the security community.

This project does not claim ownership of third-party research or resources. Original researchers and creators should be credited wherever possible.

## 🤝 Contributing

Contributions from the security community are welcome. You can suggest useful tools, write-ups, wordlists, methodologies, research, or other resources by opening an Issue or submitting a Pull Request.
